# Pavillon Libre

Prototype Godot 4.x original pour un jeu d'aventure pirate.

Le projet n'utilise aucun asset externe et ne reprend aucune marque, aucun nom, aucun modele et aucun contenu protege. La v0.1 vise seulement une base jouable propre : bateau joueur, mer de test, combat basique, ennemi et loot simple.

## Lancement

1. Ouvrir le dossier du depot avec Godot 4.x.
2. Lancer la scene principale `res://scenes/world/World.tscn`.

Godot n'est pas inclus dans ce depot.

## Controles prevus

- `Z` / `W` : avancer
- `S` : ralentir ou marche arriere legere
- `Q` / `A` : tourner a gauche
- `D` : tourner a droite
- Clic gauche : tirer a gauche
- Clic droit : tirer a droite

## Structure

- `scenes/` : scenes Godot reutilisables
- `scripts/` : scripts GDScript organises par domaine
- `docs/` : design, feuille de route et notes de production
